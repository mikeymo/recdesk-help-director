Program List Fields

 
 

Program List Fields

 
  • Program Name - Name assigned to the Program
 
  • Start Date - The first calendar date on the Program's schedule
 
  • Ages - Targeted age group (if specified)
 
  • Grades - Targeted age group (if specified)
 
  • Openings - Maximum Enrollment
 
  • Currently - Current Number of Registrants
